# Coursera Learning Platform

## About
Massive Open Online Course platform partnering with universities offering courses, certificates, and degrees with video lectures and peer grading.

## Tech Stack
- **Frontend:** Angular, TypeScript
- **Backend:** Scala, Play Framework
- **Database:** MySQL, MongoDB
- **DevOps:** AWS, Docker, Kubernetes

## Key Features
- Mooc
- Online Courses
- Certificates
- User analytics and reporting
- Mobile responsive design

## Getting Started
```bash
git clone https://github.com/coursera/platform.git
npm install
npm run dev
```
