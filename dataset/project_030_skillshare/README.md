# Skillshare Creative Platform

## About
Online learning community for creative skills offering thousands of classes with project-based learning.

## Tech Stack
- **Frontend:** React, React Native
- **Backend:** Node.js, GraphQL
- **Database:** PostgreSQL, Elasticsearch
- **DevOps:** AWS, Docker, Kubernetes

## Key Features
- Creative
- Video Courses
- Community
- User analytics and reporting
- Mobile responsive design

## Getting Started
```bash
git clone https://github.com/skillshare/platform.git
npm install
npm run dev
```
