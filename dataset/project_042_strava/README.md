# Strava Fitness Tracking

## About
Social fitness network for athletes to track runs and rides, compete on segments, and share achievements.

## Tech Stack
- **Frontend:** React, React Native
- **Backend:** Ruby, Scala
- **Database:** PostgreSQL, Cassandra
- **DevOps:** AWS, Docker, Kubernetes

## Key Features
- Fitness
- Running
- Cycling
- User analytics and reporting
- Mobile responsive design

## Getting Started
```bash
git clone https://github.com/strava/platform.git
npm install
npm run dev
```
