# Airbnb Vacation Rentals

## About
Global marketplace for vacation rentals and unique stays connecting hosts with travelers across 220+ countries.

## Tech Stack
- **Frontend:** React, React Native
- **Backend:** Ruby, Java
- **Database:** PostgreSQL, Presto
- **DevOps:** AWS, Docker, Kubernetes

## Key Features
- Vacation Rentals
- Hospitality
- Booking
- User analytics and reporting
- Mobile responsive design

## Getting Started
```bash
git clone https://github.com/airbnb/platform.git
npm install
npm run dev
```
