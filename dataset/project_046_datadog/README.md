# Datadog Infrastructure Monitoring

## About
Cloud monitoring platform providing observability across infrastructure, applications, and logs with AI-powered alerts.

## Tech Stack
- **Frontend:** React, TypeScript
- **Backend:** Go, Python
- **Database:** Cassandra, Elasticsearch
- **DevOps:** AWS, Docker, Kubernetes

## Key Features
- Monitoring
- Observability
- Apm
- User analytics and reporting
- Mobile responsive design

## Getting Started
```bash
git clone https://github.com/datadog/platform.git
npm install
npm run dev
```
