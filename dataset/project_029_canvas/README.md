# Canvas LMS

## About
Enterprise learning management system for K-12 and higher education with course management, gradebook, and analytics.

## Tech Stack
- **Frontend:** React, Redux
- **Backend:** Ruby, Rails
- **Database:** PostgreSQL, Redis
- **DevOps:** AWS, Docker, Kubernetes

## Key Features
- Lms
- Universities
- Gradebook
- User analytics and reporting
- Mobile responsive design

## Getting Started
```bash
git clone https://github.com/canvas/platform.git
npm install
npm run dev
```
