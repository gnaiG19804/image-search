# Intercom Customer Messaging

## About
Customer messaging platform with live chat, help desk, and marketing automation for businesses.

## Tech Stack
- **Frontend:** React, React Native
- **Backend:** Ruby, Rails
- **Database:** PostgreSQL, Elasticsearch
- **DevOps:** AWS, Docker, Kubernetes

## Key Features
- Customer Support
- Chat
- Messaging
- User analytics and reporting
- Mobile responsive design

## Getting Started
```bash
git clone https://github.com/intercom/platform.git
npm install
npm run dev
```
