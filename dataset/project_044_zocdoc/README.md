# ZocDoc Appointment Booking

## About
Online medical care appointment booking platform with provider search, reviews, and insurance verification.

## Tech Stack
- **Frontend:** React, React Native
- **Backend:** Python, Flask
- **Database:** PostgreSQL, Elasticsearch
- **DevOps:** AWS, Docker, Kubernetes

## Key Features
- Healthcare
- Appointments
- Booking
- User analytics and reporting
- Mobile responsive design

## Getting Started
```bash
git clone https://github.com/zocdoc/platform.git
npm install
npm run dev
```
