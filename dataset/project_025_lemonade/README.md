# Lemonade AI Insurance

## About
AI-powered insurance platform with instant claims processing through chatbot interface using behavioral economics and machine learning.

## Tech Stack
- **Frontend:** React, React Native
- **Backend:** Python, Node.js
- **Database:** PostgreSQL, MongoDB
- **DevOps:** AWS, Docker, Kubernetes

## Key Features
- Insurance
- Ai
- Claims
- User analytics and reporting
- Mobile responsive design

## Getting Started
```bash
git clone https://github.com/lemonade/platform.git
npm install
npm run dev
```
