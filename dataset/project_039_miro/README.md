# Miro Online Whiteboard

## About
Collaborative online whiteboard for distributed teams with infinite canvas and real-time collaboration.

## Tech Stack
- **Frontend:** React, TypeScript
- **Backend:** Node.js, Kotlin
- **Database:** PostgreSQL, Redis
- **DevOps:** AWS, Docker, Kubernetes

## Key Features
- Whiteboard
- Collaboration
- Brainstorming
- User analytics and reporting
- Mobile responsive design

## Getting Started
```bash
git clone https://github.com/miro/platform.git
npm install
npm run dev
```
