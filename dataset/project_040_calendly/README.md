# Calendly Scheduling Tool

## About
Automated scheduling tool eliminating back-and-forth emails with calendar integration and customizable booking pages.

## Tech Stack
- **Frontend:** React, React Native
- **Backend:** Ruby, Rails
- **Database:** PostgreSQL, Redis
- **DevOps:** AWS, Docker, Kubernetes

## Key Features
- Scheduling
- Appointments
- Calendar
- User analytics and reporting
- Mobile responsive design

## Getting Started
```bash
git clone https://github.com/calendly/platform.git
npm install
npm run dev
```
