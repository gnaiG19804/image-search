# MyFitnessPal Nutrition Tracker

## About
Calorie counter and diet tracker with barcode scanner, recipe importer, and exercise logging.

## Tech Stack
- **Frontend:** React Native, Vue.js
- **Backend:** Node.js, Python
- **Database:** PostgreSQL, MongoDB
- **DevOps:** AWS, Docker, Kubernetes

## Key Features
- Nutrition
- Calorie Tracking
- Diet
- User analytics and reporting
- Mobile responsive design

## Getting Started
```bash
git clone https://github.com/myfitnesspal/platform.git
npm install
npm run dev
```
