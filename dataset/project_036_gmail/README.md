# Gmail Email Service

## About
Web-based email service with powerful search, spam filtering, smart categorization, and 15GB free storage.

## Tech Stack
- **Frontend:** Angular, TypeScript
- **Backend:** Java, C++
- **Database:** Bigtable, Spanner
- **DevOps:** AWS, Docker, Kubernetes

## Key Features
- Email
- Search
- Spam Filter
- User analytics and reporting
- Mobile responsive design

## Getting Started
```bash
git clone https://github.com/gmail/platform.git
npm install
npm run dev
```
