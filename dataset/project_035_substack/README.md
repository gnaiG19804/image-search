# Substack Newsletter Platform

## About
Newsletter platform for independent writers with built-in payments and email delivery system.

## Tech Stack
- **Frontend:** React, Next.js
- **Backend:** Node.js, PostgreSQL
- **Database:** PostgreSQL, Redis
- **DevOps:** AWS, Docker, Kubernetes

## Key Features
- Newsletter
- Writing
- Subscription
- User analytics and reporting
- Mobile responsive design

## Getting Started
```bash
git clone https://github.com/substack/platform.git
npm install
npm run dev
```
