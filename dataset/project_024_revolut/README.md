# Revolut Digital Bank

## About
All-in-one digital banking app offering currency exchange, stock trading, cryptocurrency, and budgeting tools across 150+ currencies.

## Tech Stack
- **Frontend:** Flutter, React
- **Backend:** Java, Kotlin
- **Database:** PostgreSQL, Cassandra
- **DevOps:** AWS, Docker, Kubernetes

## Key Features
- Banking
- Fintech
- Payments
- User analytics and reporting
- Mobile responsive design

## Getting Started
```bash
git clone https://github.com/revolut/platform.git
npm install
npm run dev
```
